  1. Back wash flux Calculation
    Flux (gpm/ft2)= a+bT+cT2+dT3 where T is max back wash temperature in deg F
    Assumption: Bed expansion is 30%
    • Activated Carbon Filter, Greensand Filter (coefficients taken from GAC filter sizing spreadsheet) a = 4.649784, b = 0.035754, c = 0.001224, d = -3.54*10-6
    • Vertical /Horizontal MMF
    (coefficients taken from GAC filter sizing spreadsheet) a = 2.6229, b=0.18556, c= 0.00053 d= -2.53*10-6
  2. Straight Shell Height Calculation
    • Activated Carbon Filter
    Activated carbon media volume in ft3 = Design flow per unit in gpm (To maintain design flow per unit media volume as unity)
    Activated carbon bed height is calculated from media volume and volume of different vessel components SSH = CEILING(carbon media height*(1+freeboard/100),6)+6
    • Greensand Filter
    green sand media height is calculated from a look up table for Fe+Mn load vs green sand depth. anthracite cap height is 50% of filter area and filter area is calculated based on outer dia, shell thickness and lining thickness SSH = Max((CEILING(sum of media heights*(1+freeboard/100)-4,6)),60)
    • Vertical MMF - Media heights (sand, garnet, anthracite) are user input SSH = CEILING(sum of media heights*(1+freeboard/100),6)+6
    • Horizontal MMF – SSH = Tan length, where tan length is user input
  3. Media Volume Calculation
    • Activated Carbon Filter Activated carbon media volume in ft3 = Design flow per unit in gpm (To maintain design flow per unit media volume as unity)
    • Greensand Filter green sand volume = green sand depth *filter_area anthracite volume = filter area*0.5 (If anthracite cap is yes)
    • Vertical MMF – Cumulative volume is calculated by summing the volumes of different segments of shell. Partial volume of various media are calculated based on media depth from bottom
    • Horizontal MMF – Anthracite volume = anthracite depth* filter cross sectional area Garnet volume = garnet depth* filter cross sectional area sand volume = sand depth* filter cross sectional area
  4. Filter Cross Sectional Area Calculation
    • Activated Carbon Filter, Greensand Filter, vertical MMF in_shell_dia=outer_dia-2*(shellwall_thickness+lining_thickness) filter_area = (pi*0.25*in_shell_dia**2 )/144
    • Horizontal MMF – in_head_dia = outer_dia-2*(headwall_thickness+lining_thickness) in_shell_dia=outer_dia-2*(shellwall_thickness+lining_thickness) horizontal_area = in_shell_dia*(in_head_dia*0.25+ (SSH-2*(shellwall_thickness+lining_thickness)))/144.
  5. Test Pressure = design pressure*1.3
  6. Cleaning Sequence Calculation
    • Back wash flow (gpm) = back wash flux based on max temp* cross sectional area of filter
    • Back wash time (in mins) = max(15, ceiling({(7.48*filter cross sectional area in ft2*Straight Shell Height in inches)/(12*backwash flow)},5)) Or User can specify the back wash time if they wish to and the user specified value will over ride the calculated time.
    • Rinse flow (gpm) = round((max(design flowrate per unit*0.5,2*filter_area)/5),0) *5 {Note - The rinse flow rate should be at least 2 gpm/ ft2 of tank area, and can be as much higher as desired, up to the larger of the service flow rate or the backwash flow rate [limited by distribution design].
    The standard approach used here is to calculate the rinse rate at 50% of the Design flow rate [or 2 gpm/ ft2 of tank area, whichever is larger] as a compromise between a small rinse outlet valve and a short rinse time.”}
    • Rinse time (mins) = max(5, ceiling({(7.48*filter_area in ft2*Straight Shell Height in inches)/(12*rinse flow)},5))
    • Drain flow (gpm), If air scouring is on = round(4*filter_area/5,0)*5 …..
    {Note - Drain flow calc is based on thumb rule of 4 gpm/ft2 .
    If user specifies the drain down time, then drain down volume is calculated as per ancaster sizing calcs. Ancaster locates the drain down outlet 6” above the resting level of the media bed. The average flow (gpm) is the volume of the vessel above the drain down outlet divided by the 5 minute drain down time. The drain down line is sized for 12 ft/sec line velocity. From drain down volume and the user specified drain down time, drain down flow is calculated.}
    • Drain time (mins), = ceiling((((7.48 cross sectional area of filter *(((Straight Shell Height-total bed depth)/12)-0.5)) + (0.1309* outer diameter of filter**3*7.48/1728))/drain_flow),1) Or User can specify the drain time if they wish to and the user specified value will over ride the calculated time.
    • Air scour flow (gpm) = round(1*cross sectional area of filter*3/5,0)*5 Note : 3scm/ft2 • Air scour time (mins) =15 mins or User can specify the air scour time if they wish to and the user specified value will over ride set point of 15 mins.
    • Refill flow (gpm) = Backwash flow • Refill time (mins), = drain down flow *5 / backwash flow {Note : The refill time exploits the proportionality of the drain down flow rate and the backwash flow rate, and the known drain down time. The drain down flow rate equals the volume to be drained divided by 5 minutes. The refill step is that exact volume, except at the backwash flow rate.
    Therefore, the refill time equals the refill flow rate divided by the backwash flow rate, with the resulting dimensionless value then multiplied by 5 minutes (the original drain down time).}
    • Subsurface wash flow (gpm) = backwashflow
    • Subsurface wash time (mins) =max(10, ceiling((7.48*(Straight shell height -total bed depth)*filter cross sectional area/(12*sub surface wash flow)),5)
