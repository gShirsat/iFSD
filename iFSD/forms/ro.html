Vessel dp calculation

Average element recovery for TDS<10000 – 15%
Average element recovery for TDS>10000 – 7%
Lead element dp for TDS<10000 – 0.1 bar
Lead element dp for TDS>10000 – 0.35 bar
Total vessel dp = dp0+dp0(1-y)+dp0(1-y)2+dp0(1-y)3 +……….+dp0(1-y)n-1
Where dp0 = Lead element dp
y = average element recovery
n = no: of elements in a vessel

Assumption: - Pressure drop for an element is directly proportional to its feed flow

The default ionic rejection values have been ascertained based on several winflows runs for the afore said elements across different feed TDS and recoveries.
