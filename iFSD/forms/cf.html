1. No: of filters
= Total Flow/(Flow per TIE* No of TIE for given CF module) where TIE is Ten Inch Equivalent
2. Actual Flow per TIE
= Total Flow/(No:of filters* No of TIEs for given CF module)
3. Pressure Drop
= ((Actual Flow per TIE in gpm) *coef*viscosity)/viscosityconst
Where coef and viscosity constant are viscosity calculation parameters for given CF module.
Viscosity = 0.1 / (2.1482 * ((temp- 8.435) +(8078.4 + ((temp - 8.435)**2))**0.5) - 120.0)) *((1.0 + 0.0137* wtsolute + 0.0014 * wtsolute**2))
Where temp is in deg C and wt solute is total gmsolute/KgH2O


Note:
If CF length is less than or equal to 40”, flow per TIE should be in range of 0.9-1.4 m3/hr.
If CF length is more than 40”, flow per TIE should be in range of 0.45-0.75 m3/hr.
