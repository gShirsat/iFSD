Number of streams required to execute the unitop

Column Diameter = 4/pi*((Flow Rate / Liquid Loading Rate)0.5)*12 (inch)
NTU = log(CO2 Concentration In / CO2 Concentration Out)
HTU = a+bT+cT2+dT3 +eT4 (inch)
Packing Height = 1.25*NTU*HTU (inch)

PRO-FDD Diameter selection which satisfies below criteria:

  • Min Flow Rate < Design Flow Rate < Max Flow Rate
  • Column Clearwell Retention Time > Clearwell Retention Time

FDD Sizing Calculations

  • Column Min Flow Rate = Column Max Flow Rate / Turn Down Ratio
  • Column Max Flow Rate is specific for different column diameter based on 36 gpm/ft2
  • Clearwell Volume = ((Clearwell Height – Clearwell Freeboard)/12)*(pi/4)*((Clearwell Dia/12) 2 )*7.48
  • Clearwell Diameter = Tower Diameter
  • Tower Height = 120” (Standard Height)
  • Clearwell Height = 180” (Standard Height)
  • Clearwell Freeboard = Packing Support Grating Height + Air Inlet Duct Height + Overflow Nozzle Height + Min. Liquid Height
  • Packing Support Grating Height = 1.5” (Default or we can give option to input value)
  • Air Inlet Duct/Vent Height: (Height is specific for different Column diameters. Available form GRD 2000 Standard Fan assembly Workbook)
  • Overflow Nozzle Height = Overflow Nozzle Diameter + 3”
  • Overflow Nozzle Diameter = Input Value
  • Min. Liquid Height = 2*Suction Nozzle Clearance
  • Suction Nozzle Clearance = 4” for PRO-FDD 36,48,60,72 & 6” for PRO-FDD 84,96,108,120,132,144
  • Packing Height = Tower Height – Tower Freeboard
  • Tower Freeboard = Demister Height + Inlet Header Height + Spray Cone Height + Spray Nozzle Height
  • Demister Height = Input Value
  • Inlet Header Distributor Size = Input Value
  • Inlet Header Height = Lookup (Inlet Header Distributor Size from FRP ID & FRP OG table)
  • Spray Cone Height = Tan(30*pi/180)*Tower Diameter/2
  • Spray Nozzle Height = 5.9” (Default from Bete Catalog or we can give option to input value)
